# AT40 Viewer

A media player inspired by the Winamp Classic interface, specifically designed for streaming and viewing American Top 40 (AT40) charts.

## Features
- Winamp-inspired retro interface.
- Automatic playlist generation from AT40 chart JSON files.
- Integration with yt-dlp for streaming official music videos.
- Queue management for continuous playback.

## Prerequisites
- Python 3
- yt-dlp
- PyGObject (GTK+ 3.0)

## Installation and Usage
1. Clone the repository to your local machine.
2. Install the required dependencies:
   `pip install yt-dlp`
3. Ensure you have the necessary system libraries for GTK+ 3.0.
4. Give execution permissions to the script:
   `chmod +x viewer.py`
5. Run the application:
   `./viewer.py`
